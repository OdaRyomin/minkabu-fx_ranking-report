# みんかぶFXランキングコンテンツ テンプレート

みんかぶFXのランキング評価データをもとに、FX会社向けの半期評価レポートを作成するためのテンプレートです。

このテンプレートには実データと生成済み成果物を含めていません。利用時は `input/` にCSVなどの元データを配置し、`output/` に下書きやHTMLレポートを生成してください。

## 概要

主な用途は以下です。

- 総合順位、総合点、カテゴリ別評価を整理する
- 上位社との差分や強み・改善余地を説明する
- MarkdownまたはHTML形式で、顧客向けに説明可能な半期評価レポートを作成する
- 同一フォーマットで複数のFX会社向けレポートを作成する

## ディレクトリ構成

```text
.
├── DESIGN.md
├── README.md
├── input/
│   └── .gitkeep
├── output/
│   └── .gitkeep
└── 使用skill/
    └── build-data-driven-reports/
```

## 主なファイル・ディレクトリ

| パス | 内容 |
| --- | --- |
| `DESIGN.md` | HTMLレポートのデザイン仕様。レイアウト、配色、情報設計を定義 |
| `input/` | 元データ配置用ディレクトリ。CSV、Excel、Markdownメモなどを配置 |
| `output/` | 生成物配置用ディレクトリ。下書き、HTML、PDF化用ファイルなどを配置 |
| `使用skill/build-data-driven-reports/` | レポート作成に使用するSkill定義 |

## 入力データの例

`input/` には、レポート作成に必要なデータを配置します。

想定するデータ例は以下です。

- 総合順位・総合スコア
- カテゴリ別スコア
- FX会社の基本情報
- スプレッド関連データ
- スワップ関連データ
- レビュー関連データ
- 更新履歴・変更情報
- 編集部メモ、補足コメント

## レポート作成の流れ

1. `input/` に対象期間のデータを配置する
2. 総合順位・総合点・カテゴリ別評価を確認する
3. 更新履歴や補足メモから、対象期間中の主な変化を抽出する
4. 対象会社向けのMarkdown下書きを `output/` に作成する
5. 必要に応じてHTMLレポートを `output/` に生成する
6. `DESIGN.md` に沿って、情報順・見た目・開示範囲を確認する

## デザイン方針

HTMLレポートは、FX会社のマーケティング担当者や事業責任者が短時間で評価背景を理解できるように設計します。

主な方針は以下です。

- 冒頭で総合順位・総合点・主要KPIを提示する
- 本文では、総合評価、競争ポジション、カテゴリ診断、評価ポイント、更新事項、編集部所見の順に説明する
- 色は意味で使い、過度な装飾やカテゴリごとの多色展開は避ける
- 詳細な配点テーブル、調整係数、内部補正ロジックは公開しない

詳細は `DESIGN.md` を参照してください。

## 公開・共有時の注意

このテンプレート自体には実データを含めていませんが、利用後にGitHubなどへ公開する場合は以下を確認してください。

- `input/` 配下に社外公開できないデータが含まれていないか
- `output/` 配下に顧客名、スコア、順位、契約条件、非公開メモが含まれていないか
- 報酬単価、内部補正ロジック、詳細配点などを誤って公開していないか
- ライセンスや権利表記を明記する必要がないか
